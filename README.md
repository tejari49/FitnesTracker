# Fitness Progress Tracker Deluxe

## GitHub Pages Deployment

1. Repository nach GitHub pushen.
2. In **Settings → Pages** als Source den Branch (z. B. `main`) und Ordner `/ (root)` wählen.
3. Speichern — die Seite wird als statische Website mit `index.html`, `assets/css/styles.css` und `assets/js/app.js` ausgeliefert.

Die App nutzt `indexedDB` im Browser für lokale Speicherung (Pläne, Logs, XP).
